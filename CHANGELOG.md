## [0.0.1] - 30/05/2020

* Initial Release.
This is a smart flutter widget that is fully customizable for radio button group creations.

  Features:

     * Equal width buttons
     * radio buttons horizontal
     * radio buttons vertical
     * radio buttons equal width and height
